关于theme ​

根据 ==> vitepress 自定义的主题。详情请看官方文档。

Layout ​

配置 ​

ts
  lang: "zh-CN", // 语言
  title: "标题", // 标题
  description: "A VitePress Theme", // 描述
  head: [], // 头部标签
  markdown: { // markdown 配置
    theme: "vitesse-light", // 代码主题
    math: true, // 启用数学公式
    image: { // 图片配置
      lazyLoading: true // 懒加载
    },
    container: { // 容器配置
      tipLabel: '提示',
      warningLabel: '警告',
      dangerLabel: '危险',
      infoLabel: '信息',
      detailsLabel: '详细信息'
    },
    lineNumbers: true, // 启用行号
    config: (md) => {} // 自定义 markdown 配置
  },
  themeConfig: { // 主题配置
    giscus: true, // 是否启用 giscus (控制全局)
    menuList: [ // 菜单列表
      {
        text: "首页", // 文本
        icon: "house", // 图标
        link: ""    // 链接
      },
      ...
    ],
    aboutMe: { // 个人信息(about页面/首页)
      name: "Mtnsea", // 名称
      desc: "一个爱学习，爱分享，爱分享的人。", // 描述
      avatar: "/avatar.jpg", // 头像(可以换成链接地址)
      social: [ // 社交信息
        { icon: "github", link: "https://github.com/mtnsea" },
        { icon: "twitter", link: "https://twitter.com" },
        { icon: "weibo", link: "https://weibo.com" },
        { icon: "zhihu", link: "https://www.zhihu.com" },
      ]
    },
    footer: { text: "Mtnsea - footer - 2024/09" } // 页脚
  },

关于颜色 ​

主色根据 utils 中的 mackColor.ts 生成，使用如下：

ts
// Layout.vue
import { themeStyle } from './utils/mackColor'
onMounted(() => {
  themeStyle('#F1939C');
})

注意调用时机

函数 themeStyle 涉及到获取dom元素，所以需要在合适的时机调用。多次调用会覆盖
